# Midnite-Task

How to run the project:
Open the .xcodeproj file in Xcode and build on any simulator that is on iOS 16+ (iPhone and iPad only)


Architecture decisions:
MVVM-C, as no navigation was needed for this task, I created anything that would control navigation and presentation purely as a template if the task was to ever need extending on in the future


Any assumptions made:
I assumed that these APIs would for the most part remain stable, but I added a small amount of error handling just incase


How did you use AI to complete it:
To help sanity check data structures for the API responses

Sanity checking my approaches just incase I had done anything that was not needed or could lead to larger issues

A first form of peer reviewing to provide some on the surface feedback
